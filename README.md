# 🎉 빙고 파티 (Bingo Party)

회사 행사용 실시간 웹 빙고. 참가자는 휴대폰으로 접속, 호스트 화면은 빔프로젝터에 띄워서 진행합니다.

## 기능
- **5×5 빙고**, 숫자 1~100 랜덤 카드 (모드 1)
- **커스텀 단어 모드**: 참가자들이 단어를 입력 → 그 단어 풀에서 자동 호출 (모드 2)
- **자동 호출**: 호스트가 시작하면 설정한 간격(초)마다 자동으로 호출
- **다중 빙고 우승**: 완성 줄 수(기본 3줄) 도달 시 서버가 자동 판정 → 분쟁 없음
- 실시간 리더보드, 우승자 자동 발표, 폰 새로고침해도 카드 복구

---

## 1) 실행 (로컬)
```bash
npm install      # 최초 1회
npm start        # 서버 시작 → http://localhost:3000
```
- 호스트: `http://localhost:3000/host.html`
- 참가자: 호스트 화면의 QR 스캔 또는 코드 입력

Windows에서는 `start.bat` 더블클릭으로도 실행됩니다.

---

## 2) 인터넷에 공개하기 (참가자 폰 접속용) — 추천: Cloudflare 터널

내 PC가 서버가 되고, 터널이 외부에서 접속할 공개 https 주소를 열어줍니다. (계정/카드 불필요)
`cloudflared.exe`는 이미 이 폴더에 받아져 있습니다.

**가장 쉬운 방법 — `start-event.bat` 더블클릭**
- 서버 + 터널이 함께 실행됩니다.
- 터널 창에 뜨는 `https://xxxx.trycloudflare.com` 주소를 확인 → 거기에 `/host.html`을 붙여 브라우저로 접속.
- 예: `https://xxxx.trycloudflare.com/host.html`

**수동으로 하려면 (창 2개):**
```bash
# 창 1
npm start
# 창 2
cloudflared.exe tunnel --url http://localhost:3000
```

호스트 화면의 QR은 이 공개 주소로 자동 생성되므로, 참가자는 그냥 QR만 스캔하면 됩니다.

> ⚠️ 무료 터널 주소는 켤 때마다 **매번 새로 바뀝니다.** 그래도 호스트가 그날의 주소로
> `/host.html`을 열면 QR이 자동으로 맞춰지므로 문제 없습니다.
> PC가 켜져 있고 인터넷에 연결된 동안만 유지되며, 창을 닫으면 종료됩니다.

### 대안: 같은 WiFi (인터넷 불필요)
모두 같은 WiFi면 터널 없이도 됩니다. 내 PC의 IP 확인 후
참가자는 `http://<내IP>:3000` 으로 접속 (예: `http://192.168.0.10:3000`).
PC IP 확인: PowerShell에서 `ipconfig` → IPv4 주소.
(단, 회사 WiFi가 기기 간 통신을 막는 경우 터널 방식을 쓰세요.)

---

## 진행 순서 (당일)
1. `npm start` + 터널 실행 → 호스트가 공개 주소 `/host.html` 접속
2. 모드/간격/우승 줄수 설정 → **방 만들기** → 코드·QR 생성
3. 참가자들 QR 스캔 → 닉네임 입력 → 참가 (단어 모드면 단어 입력)
4. 인원 모이면 **게임 시작 ▶** → 자동 호출 시작
5. 우승자 자동 발표 🏆, **새 게임 준비 🔄** 로 다음 판

## 팁
- 참가 인원이 많을수록 우승 줄수를 높이면(3~4줄) 게임이 더 오래 갑니다.
- 숫자 모드 권장 호출 간격 3~4초, 단어 모드는 단어를 읽을 시간 고려해 5초+.
